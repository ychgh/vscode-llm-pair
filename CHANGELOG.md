# Change Log

All notable changes to the "vscode-llm-pair" extension will be documented in this file.

## [0.0.1] - 2025-10-31

### Added
- Initial release of VS Code LLM Pair extension
- TypeScript extension scaffold with proper build configuration
- LLM provider interface for extensibility
- OpenAI provider implementation with streaming support
- Sidebar webview chat interface
- Configuration options for API keys, models, and endpoints
- Conversation history management
- Real-time streaming responses
- Error handling and user feedback
- Extension activation commands
- VS Code launch and task configurations
